
instructions from: https://github.com/vadimdemedes/google-images
and here: https://developers.google.com/custom-search/json-api/v1/overview
https://developers.google.com/custom-search/json-api/v1/reference/cse/list

NOTE: only accepts 100 searches per day. Then $5/1000 additional searches.

1. Create a Google Custom Search Engine (https://cse.google.com/cse)
2. name the search, but do NOT specify any sites.
3. under advanced, enable image search.
4. under advanced, 'Restrict Pages by Schema.org Types'. Use 'Thing' as the limitation.
5. get CX ID from this page.
5. Set up a Google Custom Search Engine API (Set up a Google Custom Search Engine API -- https://console.developers.google.com/)
6. Create a Credential (API Key).
7. check quota usage here: https://code.google.com/apis/console
